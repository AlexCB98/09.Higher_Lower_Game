# Higher Lower Game (Python)

A simple command-line game where you guess which account has more followers.

This project was built as part of my Python learning journey, focusing on dictionaries, functions, loops, and program structure.

---

## How it works

- You are shown two accounts:
  - Account A
  - Account B
- Each account has:
  - Name
  - Description
  - Country

Your task:
Guess which account has more followers.

- Type `A` or `B`
- If correct → your score increases
- If wrong → game ends and final score is shown
